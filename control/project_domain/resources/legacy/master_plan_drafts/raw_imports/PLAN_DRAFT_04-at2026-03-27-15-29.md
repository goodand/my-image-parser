# Presentation Image Mapping Extension Draft

## Request Summary

프레젠테이션 자료에서 이미지 자료를 추출하고 싶다.

기본 흐름:

presentation file을 다운로드
-> presentation file에서 이미지 추출
-> 이미지에 캡션 달기
-> 이미지와 이미지 캡션 기반 presentation file 생성

이미 완료된 프로세스:

- presentation file에서 이미지 추출
- Gemini로 이미지에 캡션 달기
- 이미지와 이미지 캡션 기반 presentation file 생성

추가로 필요한 절차:

- 이미지를 벡터 임베딩
- 리랭커를 사용해 관련 문서 매핑 후보 리스트 출력
- 이후 사람 검토 수행

사람이 할 일 5가지:

1. 이미지에 캡션이 잘 달렸는지 검토
2. 이미지 캡션이 비적합하면 직접 수정
3. 리랭커 Top-5 이미지 관련 문서 매핑 순위 검토
4. Top-5 중 최종 매핑 결정 또는 이상치 라벨 설정
5. 이상치 라벨 이미지에 대한 설명서를 직접 작성
