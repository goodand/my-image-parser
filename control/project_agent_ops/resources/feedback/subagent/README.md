# Subagent Feedback

Use this bucket for reusable feedback about subagent behavior in this workspace.

Examples:

- packet quality feedback
- overlap or write-boundary issues
- missing context patterns
- retry guidance for failed worker runs
- notes about thread caps, model selection, or MCP usage during subagent execution

Promotion rule:

- if the note becomes a reusable workflow recipe, promote it to `skill_candidates/repeated_tasks/`
- if the note becomes a reusable guardrail or failure pattern, promote it to `skill_candidates/repeated_issues/`
