# Repeated Issues

Canonical bucket for repeated issues that recur across runs, skills, MCP usage, or agent execution.

Use this directory when the same failure or friction pattern appears often enough to justify a checklist, guardrail, or new skill.

Typical sources:

- provider feedback that reveals the same failure pattern
- subagent execution friction seen across multiple runs
- MCP boot or configuration drift that keeps returning
- repeated troubleshooting outcomes from smoke reports
